# === P U R E - A R C H ===
A security focused  Arch Linux Operating System installer using python.

## Highlights
- [ ] **systemd** focused
- [ ] **btrfs** file system
- [ ] **luks** (luks2) disk enxryption
- [ ] **no boot-loader** like *grub*, *systemd_boot*, etc
- [ ] **configurable** with a toml configurations file
- [ ] **hardened**
